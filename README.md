# Source code for Fancy Player

This mod allows you to color your particles and dash fire, overriding the game's default colors. Everything is inside the mod settings.

Serving suggestion: I match the dash fire and particle colors with my glow color

## Features

* Override startColor and finishColor of player particles
** Rainbow player particles
* Override the color of the dashing fire
** Rainbow dash fire
* Adjust RGB speed
* Scale radius emitter maxRadius with player size
